# 佐证材料清单 — 附件5映射

## 说明

本文件对照省专精特新申报系统"附件5：专精特新中小企业佐证材料"，将其每一项映射到系统 `initMaterials()` 中初始化的材料清单分类体系。

> 附件5原文是供参考的清单，企业只需证明符合认定条件即可，不一定每一项都提供。
> 系统当前材料清单基于附件5 + 专精特新认定标准设计，实际申报时以系统要求为准。

## 逐项说明

### 1. 目录
- 系统材料清单中无需单独上传目录
- PDF 打包时由系统自动生成或用户自行编排

### 2. 营业执照
- **系统材料**: `营业执照（副本）` — 基础资质 / 必交
- **OCR 入口**: `POST /api/declarations/{id}/ocr/business-license`
- **OCR 提取字段**: companyName, creditCode, legalRepresentative, capitalAmount, establishedDate, fullAddress, province, city, district
- **关联表单字段**: `f_name`, `f_creditCode`, `f_legalRep`, `f_regCapital`, `f_estDate`, `f_address` + 省/市/区下拉框
- **参考映射**: `attachment_2_license`

### 3. 经审计的年度审计报告（2021/2022/2023）
- **系统材料**: `近两年财务审计报告` — 财务指标 / 必交
- 附件5要求 2021-2023 三个年度，系统初始化为"近两年"，视申报批次可调整
- 若2023年审计报告未出，可先用盖章财务报表替代
- **参考映射**: `attachment_3_audit_report`

### 4. 研发投入佐证材料
- **系统材料**: `近三年研发费用专项审计报告` — 财务指标 / 必交
- 如年度审计报告中已体现研发费用数据，可不重复提供
- **参考映射**: `attachment_4_rd_expense`

### 5. 股权融资佐证（非必须）
- **条件**: 上年度营收 < 1000 万元时提供
- 近两年新增股权融资总额（合格机构投资者实缴额）的佐证材料
- 系统表单中对应 `equity_financing_2023/2024/2025` 字段，不单独作为材料上传项
- **参考映射**: `attachment_5_equity_financing`

### 6. 国家级/省级科技奖励证书（非必须）
- 近三年获得的国家级/省级科技奖励
- 系统在 `awards` 表中以 JSON 存储（`national_award`, `provincial_award`）
- 不单独作为材料上传项，但可归入"其他相关佐证材料"统一上传
- **参考映射**: `attachment_6_awards`

### 7. 创客中国500强（非必须）
- 系统表单字段: `maker_china_top50`
- 不单独作为材料上传项
- **参考映射**: `attachment_7_maker_china`

### 8. 数字化水平自测结果截图
- **条件**: 必交项
- 培育平台上的数字化水平自测结果截图
- 系统当前材料清单中无此独立项，建议在**经营规范**分组下补充材料项：
  - `数字化水平自测结果截图` — 经营规范 / 必交
- **参考映射**: `attachment_8_digitalization`

### 9. 质量管理水平证明材料
- **系统材料**: `管理体系认证证书` — 经营规范 / 必交
- 涵盖: ISO9001 等管理体系认证、自主品牌、参与制修订标准等
- 系统表单中 `management_certifications`（MultiSelect）配合
- 品牌相关材料归入`自有品牌证明及市场占有率报告`
- **参考映射**: `attachment_9_quality_mgmt`

### 10. 本省重点产业领域情况说明
- 系统无单独材料项
- 可在申报表单中通过 `industry_chain`, `leading_product_category`, `industrial_six_basics` 等字段体现
- 作为其他相关佐证材料上传（归入"特色化"分组）
- **参考映射**: `attachment_10_key_industry`

### 11. 创新生产经营情况说明及证明
- 系统无单独材料项
- 对应表单: `company_profile`（企业总体情况介绍）
- 相关证明材料归入"特色化"或"其他相关佐证材料"
- **参考映射**: `attachment_11_innovation_operation`

### 12. 主导产品细分市场占有率证明
- **系统材料**: `自有品牌证明及市场占有率报告` — 特色化 / 必交
- 允许企业自证，需体现计算过程和原始数据来源
- 系统表单字段: `domestic_share_2024/2025`, `international_share_2024/2025`, `market_share_description`
- **参考映射**: `attachment_12_market_share`

### 13. 上市进展情况说明及证明
- 系统在 `listing_status`, `stock_code`, `listing_progress`, `planned_listing_exchange` 等表单字段体现
- 不单独作为材料上传项
- **参考映射**: `attachment_13_listing`

### 14. 知识产权证明材料
- **系统材料**: `知识产权证书` — 创新能力 / 必交
- 分类:
  - **I类知识产权**: 发明专利、植物新品种、国家级农作物品种、国家新药、国家一级中药保护品种、集成电路布图设计专有权（不包含转让未满1年的）
  - **I类高价值知识产权**: 海外同族专利/收入、维持超10年、高质押融资额、获国家科技奖/专利奖的I类
  - **II类知识产权**: 软件著作权、授权后维持超2年的实用新型/外观设计（不包含转让未满1年的）
- 系统字段: `class1_ip_count`, `invention_patent_count`
- **参考映射**: `attachment_14_ip`

### 15. 研发人员占比证明材料
- **系统材料**: `研发人员花名册及社保证明` — 创新能力 / 选交
- 要求: 申报年度12月底缴纳社保人数证明 + 研发人员花名册
- 表单字段: `employee_count_2023/2024/2025`, `rd_employee_count_2023/2024/2025`, `rd_employee_ratio_2023/2024/2025`
- **参考映射**: `attachment_15_rd_staff`

### 16. 建立研发机构证明材料
- **系统材料**: `研发机构证明材料` — 创新能力 / 必交
- 形式: 证书、奖牌、文件、机构规章制度、实景图片等
- 系统表单: `rd_institutions`（JSON）
- **参考映射**: `attachment_16_rd_institution`

### 17. 公共信用报告 / 无违法违规记录证明
- **系统材料**: `无违法记录承诺函` — 经营规范 / 必交
  - `近两年无重大安全/环保事故证明` — 经营规范 / 必交
- 附件5要求 36 个月公共信用报告，访问信用中国获取
- 系统当前使用企业自行出具的承诺函，建议用户仍按附件5要求从信用中国下载官方报告
- **参考映射**: `attachment_17_credit_report`

### 18. 申报材料真实性声明
- 系统导出申报表 → 法人签字、企业盖章 → 再扫描上传
- 打包在最终 PDF 中
- **参考映射**: `attachment_18_authenticity_declaration`

### 19. 其他相关佐证材料
- 与填报内容对应的其他佐证材料
- 各分组下现有材料项未覆盖的补充项
